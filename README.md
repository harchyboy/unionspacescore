# UNION Spaces Core

A collection of HTML pages for property and unit management in UNION Spaces.

## Files

- **Add Unit To Property Page.html** - Interface for adding units to properties
- **Properties Detailed Card.html** - Detailed property card view
- **Property List Dashboard.html** - Main property list dashboard
- **Property List Dashboard Add Po.html** - Property list dashboard with PO addition functionality
- **Units Details Page.html** - Detailed unit information page
- **Units More Detail Card.html** - Extended unit detail card view

## Getting Started

These are standalone HTML files that can be opened directly in a web browser.
